# Monitor.it

This Is A Lightweight SwiftUI App That Allows You To Track Health Data And Start Workouts From Apple Watch.

## **Features**

- Start Workouts
- View Activity Data
- View Heart Rate Data
- Play Music During Workouts
- Breathe Fuctionality
